Калибровка камеры на роботе

1. Подключить к Rasberry монитор и клавиатуру.

2. В консоли запустить vcap.sh из каталога /ros/src/yy_ctl/ctl/scripts/aruco.
   Эта утилита выводит на экран в отдельное окно видеопоток с камеры и осуществлет захват видео 
   и создание снимков с камеры по нажатию клавиши "пробел". Утилита формирует снимки 
   для калибровки камеры. В качестве основы используется обычно щахматная доска 8х8.

3. Снимки имеют расширение, которое устанавливается в утилите vcap.py (сейчас - png).

4. Параметры vcap.py:
   -d - устройство, с которого берется видеопоток (/dev/video0)
   -c - количество колонок в изображении (640)
   -r - количество строк в изображении (480)
   -o - каталог, в который сохраняются полученные снимки (./images/calibrate)

5. Создание очередного снимка - "пробел", выход - Esc.

6. Шахматную доску надо располагать перед камерой так, чтобы она полностью попадала в объектив.
   Это можно контролировать с помощью окна вывода видеопотока.
   Надо сделать минимум 10 снимков с разных ракурсов - прямо и с поворотом вокруг разных осей.

7. Запустить утилиту создания калибровочной матрицы:
   calibr.sh

8. Параметры calibr.py:
   -r - количество строк в изображении (пикселей по вертикали, 480)
   -с - количество столбцов в изображении (пикселей по горизонтали, 640)
   -o - каталог, в который сохраняются созданные файлы, и имя файла (./cfg/outp)
   --img - каталог, из которого берутся снимки для калибровки (./images/calibrate/*.png)

9. Если все снимки распознаны удачно, то полученные файлы можно переименовать (в camera и camera.yaml).
